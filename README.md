# Web-Utils

** Helpful TS/JS utilities **

** example **

```
import { Utils } from @pastaboss/web-utils 

Utils.strings.replaceUnderscore('my_string');

//result: 'my string' 

Utils.strings.removeUnderscore('my_string');

//result: 'mystring' 

or:

console.log(Utils.numbers.multiplyDecimals(0.1, 0.1));

```
